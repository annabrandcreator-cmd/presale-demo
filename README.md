# ИИ-инженер по продажам · демо-кейс

Вымышленная B2B-ниша **«ПромАэро Системы»** (промышленная вентиляция) для демонстрации на сайте и в портфолио.  
Реальная компания не используется — каталог, цены и ИНН вымышленные.

## Что показывает демо

1. **Техническая квалификация** — пошаговый сбор параметров объекта в чате  
2. **Подбор оборудования** — расчёт расхода воздуха, выбор ПВУ, фильтров, воздуховодов  
3. **Генерация КП** — PDF коммерческого предложения  
4. **Дашборд сделки** — этапы контура (как на кейсе rocai.ru, но без привязки к бренду клиента)

## Быстрый старт (локально)

```bash
cd autonomous-sales-engineer-demo
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Откройте:

- **Демо-чат:** http://localhost:8080  
- **Дашборд:** http://localhost:8080/dashboard  
- **Health:** http://localhost:8080/health  

На macOS для WeasyPrint могут понадобиться системные библиотеки — проще через Docker (ниже).

## Docker

```bash
docker compose up --build
```

Демо будет на http://localhost:8080

## Деплой (Railway / Render)

1. Залейте папку в GitHub-репозиторий  
2. Подключите к Railway → Deploy from GitHub  
3. Variables: `TEST_MODE=1`, `PORT=8080`  
4. Volume на `/data` (опционально, для сохранения сделок)  
5. Публичный домен → вставьте в кейс на сайте  

## Структура

```
app.py              Flask API + раздача статики
engine.py           квалификация и подбор (rule-based)
generate_kp.py      PDF коммерческого предложения
catalog.json        вымышленный каталог и вопросы
static/             лендинг, чат, дашборд
```

## Сценарий для записи видео / кейса

1. Нажать «Запустить демо-сделку»  
2. Пройти 7 вопросов (цех 1200 м², высота 6 м, −28°C, G4…)  
3. Открыть «Контур сделки» — увидеть этапы  
4. Скачать PDF КП  

## Для страницы кейса на сайте

Рекомендуемые формулировки:

- **Ниша:** сложный B2B-пресейл (промышленная вентиляция), демо-бренд  
- **Проблема:** 2 дня на смету, ошибки в подборе, нет ответа ночью  
- **Решение:** автономный контур заявка → квалификация → расчёт → КП  
- **Метрики демо:** ~3 мин вместо 48 ч (в тестовом контуре)  
- **CTA:** ссылка на живое демо + «Обсудить внедрение»
